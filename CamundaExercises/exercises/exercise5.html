Add User Tasks

Goal

Add a User Task to review the tweet to your process and test it within your JUnit test. Use the Java API to control the task work.

Short description

  • Change the manual tasks to user tasks.
  • Add the tasks to the management group.
  • Extend the JUnit test to query for the user task and complete it.
  • Assert the expected state of the process instance and the tasks.

Detailed steps

  1. Morph the manual task(s) into a user task(s).
  2. Set the candidate-groups of the review task to management.
  3. Improve the test case from the last lab to query tasks from the task list (maybe add some assertions) and complete this tasks via the API (see Java Docs).
  4. Change the test case to create the “approved” process variable when completing the task, and not already when starting the process instance.
  5. Run the test case and check for the green bar.
